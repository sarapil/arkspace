## يعمل مع ERPNext الحالي لديك

ARKSpace لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/arkspace
bench --site your-site install-app arkspace
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
