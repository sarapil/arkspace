## 1. الإعدادات الأساسية

الإعدادات المركزية والأدوات المساعدة للتطبيق بالكامل.

### القدرات

- **إعدادات أرك سبيس** (مستند فردي): معلومات الشركة، العملة، المنطقة الزمنية، تنسيق التاريخ، تفضيلات اللغة
- بادئات قابلة للتكوين للحجوزات (`BK`)، العضويات (`MEM`)، الفواتير، العملاء المحتملين (`WL`)
- مفاتيح تبديل الميزات: VoIP، ARKAMOR IoT، سوق ARKANOOR، ميزات الذكاء الاصطناعي
- بيانات اعتماد التكامل: FreePBX، OpenAI، ARKANOOR API

### الأدوار

جميع الأدوار يمكنها قراءة الإعدادات؛ فقط **مدير أرك سبيس** يمكنه التعديل.

## 2. إدارة المساحات

إدارة مخزون مساحات العمل المشتركة، الأنواع، المرافق، والحجوزات.

### القدرات

- **نوع المساحة**: تعريف الفئات (مكتب مشترك، مكتب مخصص، مكتب خاص، غرفة اجتماعات، قاعة فعاليات، مكتب افتراضي) مع أيقونات وألوان ومفاتيح حجز
- **المرفق**: تعريف الخدمات الإضافية مع تسعير بالساعة/يومي/شهري وعلامة مجاني
- **مساحة العمل المشتركة**: وحدات فردية باسم (ثنائي اللغة)، النوع، الفرع، الطابق، السعة، المساحة، الأسعار، تتبع الحالة، جدول المرافق، معرض الصور
- **حجز المساحة** (قابل للاعتماد): دورة حجز كاملة — قيد الانتظار ← مؤكد ← تم الوصول ← تم المغادرة، مع دعم الإلغاء/عدم الحضور
- **تسجيل الوصول/المغادرة**: حالة لحظية مع أحداث `space_status_changed` في الوقت الفعلي
- **مخطط الطابق**: مخطط طابق تفاعلي بصري مع ألوان حسب الحالة، نافذة حجز سريع، إحصائيات إشغال مباشرة
- **العمليات الجماعية**: تسجيل وصول/مغادرة/إلغاء/عدم حضور جماعي من عرض القائمة
- **المهام التلقائية بالساعة**: تحديد عدم الحضور (مهلة ساعتين)، تسجيل مغادرة تلقائي للحجوزات المنتهية

### واجهات برمجة التطبيقات

| النقطة النهائية | الوصف |
|---------------|-------|
| `get_available_spaces` | استعلام المساحات حسب النوع والفرع والتاريخ |
| `create_booking` | إنشاء واعتماد حجز |
| `check_in` | تسجيل الوصول لحجز مؤكد |
| `check_out` | تسجيل المغادرة من حجز |

## 3. إدارة العضويات

إدارة الاشتراكات مع خطط ومحافظ رصيد وأتمتة دورة الحياة.

### القدرات

- **خطة العضوية**: 6 أنواع خطط مع مستويات تسعير (شهري/ربع سنوي/سنوي)، رسوم تأسيس، ساعات مشمولة، أرصدة، ضيوف، ساعات غرف اجتماعات، صفحات طباعة، تخزين
- **العضوية** (قابلة للاعتماد): دورة حياة كاملة — مسودة ← نشط ← منتهي ← ملغى ← موقف، مع خيار التجديد التلقائي
- **محفظة رصيد العضو**: محفظة لكل عميل مع معاملات إضافة/خصم/انتهاء/استرداد
- **التجديد التلقائي**: مهمة يومية تجدد العضويات المؤهلة
- **تذكيرات الانتهاء**: رسائل بريد إلكتروني تلقائية قبل 7 أيام ويوم واحد من الانتهاء
- **لوحة تحكم العضو**: نقطة نهائية موحدة تعرض العضويات والحجوزات والمحفظة والإحصائيات

### المهام المجدولة

- `check_membership_expiry` — يومياً: تحديد العضويات المنتهية
- `auto_renew_memberships` — يومياً: تجديد تلقائي للمؤهلين
- `send_membership_expiry_reminders` — يومياً: تذكيرات بالبريد الإلكتروني

## 4. إدارة علاقات العملاء

إدارة العملاء المحتملين وخط أنابيب المبيعات لتحويل المهتمين إلى أعضاء.

### القدرات

- **العميل المحتمل**: تتبع المهتمين مع المصدر (الموقع، زيارة مباشرة، إحالة، وسائل التواصل، حدث، شريك)، حالة خط الأنابيب (جديد ← تم التواصل ← جولة مجدولة ← تفاوض ← تم التحويل ← خسارة)
- **جولة في المساحة**: جدولة وتتبع جولات المنشأة مع التقييم وتتبع النتائج والتحويل التلقائي
- **التحويل لعميل**: تحويل العميل المحتمل بنقرة واحدة ينشئ سجل عميل ويربطه بالعضوية
- **جدولة جولة**: إنشاء جولة وتحديث حالة العميل المحتمل تلقائياً

## 5. العقود والمستندات

إدارة المستندات القانونية، قوالب العقود، وإيصالات الدفع.

### القدرات

- **قالب العقد**: قوالب مدعومة بـ Jinja للعضوية، الحجز، إيجار المكتب، قاعة الفعاليات، المكتب الافتراضي؛ يدعم العربية والإنجليزية وثنائي اللغة
- **المستند القانوني**: تخزين وتتبع مستندات الأعضاء القانونية (البطاقة الشخصية، جواز السفر، السجل التجاري، البطاقة الضريبية، إلخ)
- **عقد العضو** (قابل للاعتماد): دورة حياة كاملة — مسودة ← نشط ← منتهي ← منهي ← ملغى؛ التوقيعات والشهود
- **إيصال الدفع** (قابل للاعتماد): 6 أنواع إيصالات ثنائية اللغة مع طرق دفع متعددة

### تنسيقات الطباعة

- تأكيد الحجز
- بطاقة العضوية
- إيصال العضوية

## 6. التدريب والشارات

نظام إدارة التعلم مع وحدات وجلسات وشارات وتتبع التقدم.

### القدرات

- **الوحدة التدريبية**: مصنفة (تقنية، أعمال، إبداعية، صحة، مجتمع، تهيئة) بمستويات (مبتدئ، متوسط، متقدم)
- **الجلسة التدريبية** (قابلة للاعتماد): جلسات مجدولة مع مكان (حضوري أو عبر الإنترنت)، مدرب، إدارة الرسوم
- **الشارة التدريبية**: تلعيب بفئات شارات (إتمام، سلسلة، إتقان، مجتمع، خاصة) ومستويات (برونزي، فضي، ذهبي، بلاتيني)
- **تقدم تدريب المستخدم**: تتبع تقدم فردي لكل مستخدم لكل وحدة، منح شارات تلقائي عند الإتمام

## 7. التكاملات

جسور بين أرك سبيس و ERPNext للفوترة وإدارة الموظفين.

### القدرات

- **جسر الفوترة**: إنشاء فاتورة مبيعات تلقائية عند اعتماد/إلغاء الحجز والعضوية
- **ربط الموظفين**: ربط تلقائي للموظفين بالعملاء للوصول لبوابة الأعضاء
- **حالة التكاملات**: تقرير بالتطبيقات المثبتة (ERPNext، HRMS، Payments)
- **الفواتير غير المدفوعة**: استرجاع الفواتير المستحقة لعضو

## 8. التوثيق التلقائي

نظام توثيق يولّد تلقائياً للتطبيق نفسه.

### القدرات

- **إدخال التوثيق**: مستندات منظمة للوحدات، أنواع المستندات، واجهات البرمجة، سير العمل، الدروس، الأسئلة الشائعة — ثنائي اللغة مع أمثلة كود
- **المولد التلقائي**: مهمة Cron ليلية (2 صباحاً) تعيد توليد إدخالات التوثيق من فحص الكود
- **مولد README**: إنشاء/تحديث تلقائي لملفات README لأنواع المستندات عند الإدراج/التحديث

## 9. نظام التصميم

تكوين السمات والأيقونات ودعم RTL.

### القدرات

- **إعدادات التصميم** (مستند فردي): تحكم مركزي بالسمة — ألوان أساسية/ثانوية/تمييز/نجاح/خطر، تنسيق الأزرار، تنسيق الروابط، مكتبة الأيقونات، تبديل RTL، خطوط عربية/إنجليزية
- **نظام تصميم CSS**: متغيرات CSS مخصصة، تخطيطات متوافقة مع RTL
- **مساعدات Jinja**: `get_icon()` و `get_color()` متاحة في القوالب
- **ألوان العلامة التجارية**: كحلي (#1B365D) أساسي، ذهبي (#C4A962) تمييز
