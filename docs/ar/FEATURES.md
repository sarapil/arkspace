## 1. النواة — التهيئة المركزية

- **إعدادات ARKSpace** (مستند فردي): الشركة، العملة، المنطقة الزمنية، اللغة، البادئات
- مفاتيح تشغيل/إيقاف للميزات: VoIP, IoT, السوق, الذكاء الاصطناعي
- إدارة بيانات اعتماد التكامل
- دوال مساعدة مشتركة عبر الوحدات

## 2. المساحات — إدارة المساحات

- **6 أنواع مساحات**: مكتب مشترك، مكتب مخصص، مكتب خاص، غرفة اجتماعات، مساحة فعاليات، مكتب افتراضي
- **مخطط طوابق تفاعلي**: حالات بالألوان، حجز سريع، تصفية حسب الطابق
- **ARK Live**: صفحة مراقبة المساحات بالوقت الفعلي
- **إدارة الخدمات الإضافية**: تسعير بالساعة/يومي/شهري، علم مجاني
- **حجز المساحة** (قابل للإرسال): دورة حياة كاملة — قيد الانتظار → مؤكد → تم الدخول → تم الخروج
- **عمليات جماعية**: تسجيل دخول/خروج/إلغاء/عدم حضور جماعي
- **تحديثات فورية**: أحداث `realtime` عند كل تغيير حالة
- **مهام تلقائية**: كشف عدم الحضور (فترة سماح ساعتين) وخروج تلقائي (كل ساعة)

## 3. العضويات — منصة الاشتراكات

- **6 أنواع خطط** بأسعار متدرجة (شهري/ربع سنوي/سنوي)
- **محفظة الرصيد**: نظام رصيد مسبق الدفع لكل عضو مع سجل معاملات
- **تجديد تلقائي**: مهمة يومية للعضويات المؤهلة
- **تذكيرات الانتهاء**: بريد إلكتروني قبل 7 أيام ويوم واحد
- **واجهة لوحة العضو**: نقطة نهاية موحدة للعضويات والحجوزات والمحفظة والإحصائيات

## 4. إدارة العملاء — خط المبيعات

- **تتبع العملاء المحتملين**: 6 قنوات اكتساب (موقع إلكتروني، زيارة، إحالة، وسائل التواصل، فعالية، شريك)
- **خط المبيعات**: جديد → تم التواصل → جولة مجدولة → تفاوض → تم التحويل / خسارة
- **جدولة الجولات**: إنشاء جولة مساحة عمل، تحديث حالة العميل تلقائياً
- **تحويل بنقرة واحدة**: عميل محتمل → عميل → جاهز للعضوية

## 5. العقود — الإدارة القانونية

- **قوالب Jinja**: توليد عقود بالعربية والإنجليزية وثنائية اللغة
- **المستندات القانونية**: 10 أنواع مستندات مع رفع الوجه/الخلف، تتبع الصلاحية
- **عقود الأعضاء** (قابلة للإرسال): توقيعات رقمية، شاهد، موقع الشركة
- **إيصالات الدفع** (قابلة للإرسال): 6 طرق دفع، تتبع الفترات
- **تنسيقات الطباعة**: تأكيد الحجز، بطاقة العضوية، الإيصال

## 6. التدريب — منصة التعلم

- **وحدات تدريبية**: مصنفة (تقني، أعمال، إبداعي، صحة، مجتمع، تهيئة) بمستويات
- **جلسات تدريبية** (قابلة للإرسال): مجدولة بمكان ومدرب وسعة ورسوم
- **التلعيب**: نظام شارات (برونزي، فضي، ذهبي، بلاتيني) عبر 5 فئات
- **تتبع التقدم**: لكل مستخدم لكل وحدة مع منح شارات تلقائي

## 7. التكاملات — جسر ERPNext

- **فوترة تلقائية**: فاتورة مبيعات عند إرسال الحجز/العضوية
- **أرصدة الإلغاء**: التعامل مع الفوترة عند الإلغاء
- **ربط الموظفين**: ربط تلقائي بين سجلات الموظف ↔ العميل
- **حالة التكامل**: تقرير التطبيقات المثبتة (ERPNext, HRMS, Payments)
- **الفواتير غير المدفوعة**: واجهة لاستعلام الأرصدة المستحقة

## 8. التوثيق — توثيق ذاتي

- **مدخل التوثيق**: مستندات منظمة للوحدات والمستندات والواجهات وسير العمل والدروس والأسئلة الشائعة
- **مولد تلقائي**: مهمة ليلية تعيد توليد التوثيق من فحص الكود
- **مولد README**: إنشاء تلقائي لـ README المستندات عند الإدراج/التحديث

## 9. التصميم — السمات والعلامة التجارية

- **تهيئة التصميم**: ألوان أساسية/ثانوية/تمييز، تنسيق الأزرار/الروابط
- **مكتبة الأيقونات**: Font Awesome 6, Material Icons, أو مخصص
- **دعم RTL**: تبديل كامل لاتجاه النص من اليمين لليسار
- **تهيئة الخطوط**: خطوط منفصلة للعربية والإنجليزية
- **مساعدات Jinja**: `get_icon()` و `get_color()` للقوالب
- **ألوان العلامة**: كحلي (#1B365D) أساسي، ذهبي (#C4A962) تمييز
