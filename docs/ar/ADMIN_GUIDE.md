## التثبيت والإعداد

### المتطلبات

- Frappe v16 (أو v15)
- ERPNext (تابع مطلوب)
- Python 3.12+, MariaDB 10.6+, Redis 6+

### التثبيت

```bash
bench get-app https://github.com/arkan/arkspace.git
bench --site your-site install-app arkspace
bench --site your-site migrate
bench build --app arkspace
```

### معالج الإعداد

عند الوصول لأول مرة، يرشدك معالج الإعداد المكون من 4 مراحل:

1. **معلومات الشركة** — الاسم، العملة، المنطقة الزمنية
2. **إعداد المساحات** — تعريف أنواع المساحات والمساحات الأولية
3. **خطط العضوية** — إعداد خططك الأولى
4. **التصميم** — الشعار، الألوان، العلامة التجارية

## الإعدادات

### إعدادات ARKSpace

انتقل إلى **ARKSpace > الإعدادات** (أو ابحث عن "إعدادات ARKSpace").

| الإعداد | الوصف | الافتراضي |
|---------|-------|-----------|
| الشركة | شركة ERPNext للفوترة | (أول شركة) |
| العملة | العملة الافتراضية | USD |
| تنسيق التاريخ | تنسيق العرض | dd-mm-yyyy |
| المنطقة الزمنية | منطقة الخادم | UTC |
| اللغة الافتراضية | ar أو en | en |
| بادئة الحجز | بادئة معرف الحجز | BK |
| بادئة العضوية | بادئة معرف العضوية | MEM |
| بادئة الفاتورة | بادئة الفاتورة | INV |
| بادئة العميل المحتمل | بادئة المعرف | WL |

### مفاتيح الميزات

| الميزة | الوصف |
|--------|-------|
| تفعيل VoIP | تكامل الهاتف عبر الإنترنت |
| تفعيل ARKAMOR IoT | تكامل أجهزة إنترنت الأشياء |
| تفعيل ARKANOOR | تكامل السوق الإلكتروني |
| تفعيل ميزات AI | تحليلات بالذكاء الاصطناعي |

## إدارة المستخدمين والأدوار

### الأدوار المخصصة

| الدور | المستخدم النموذجي | الصلاحيات الرئيسية |
|-------|-------------------|-------------------|
| مسؤول ARKSpace | مسؤول النظام | وصول كامل لجميع المستندات |
| مدير ARKSpace | مدير العمليات | إنشاء، قراءة، تعديل، إرسال |
| مبيعات ARKSpace | فريق المبيعات | وصول CRM + قراءة المساحات |
| عمليات ARKSpace | موظف العمليات | إدارة الحجوزات/المساحات |
| استقبال ARKSpace | الاستقبال | عمليات تسجيل الدخول/الخروج |
| عضو ARKSpace | العميل | قراءة سجلاته فقط |
| مشاهد ARKSpace | أصحاب المصلحة | قراءة فقط |

### تعيين الأدوار

1. انتقل إلى قائمة **المستخدم**
2. افتح المستخدم
3. في قسم **الأدوار**، حدد دور ARKSpace المناسب
4. احفظ

### أمان على مستوى الصف

الأعضاء بدور "عضو ARKSpace" يرون تلقائياً فقط:
- حجوزات المساحات الخاصة بهم
- عضوياتهم
- محافظ الرصيد الخاصة بهم

يتم فرض هذا على مستوى استعلام قاعدة البيانات — لا حاجة لإعدادات إضافية.

## إعداد المساحات

### 1. تعريف أنواع المساحات

انتقل إلى **المساحات > نوع المساحة > + جديد**

الأنواع المتاحة: مكتب مشترك، مكتب مخصص، مكتب خاص، غرفة اجتماعات، مساحة فعاليات، مكتب افتراضي.

### 2. تعريف الخدمات الإضافية

انتقل إلى **المساحات > خدمة إضافية > + جديد**

حدد: الاسم، السعر بالساعة/يومي/شهري، مجانية أم لا.

### 3. إنشاء المساحات

انتقل إلى **المساحات > مساحة عمل مشتركة > + جديد**

الإعدادات:
- الاسم (بالعربية والإنجليزية)
- النوع، الفرع، الطابق
- السعة، المساحة (م²)
- الأسعار بالساعة، يومياً، شهرياً
- الحالة (متاح، مشغول، صيانة، محجوز)
- الخدمات الإضافية المرتبطة
- معرض الصور

## سير العمل

يتضمن ARKSpace 3 سير عمل مُعدة مسبقاً:

### الموافقة على الحجز
```
مسودة → قيد الانتظار → مؤكد → تم الدخول → تم الخروج
                     ↘ ملغي
                     ↘ عدم حضور
```

### دورة حياة العضوية
```
مسودة → نشط → منتهي
          ↘ معلق
          ↘ ملغي
```

### خط المبيعات
```
جديد → تم التواصل → جولة مجدولة → تفاوض → تم التحويل
                                          ↘ خسارة
```

## المهام المجدولة

| المهمة | التكرار | الوظيفة |
|--------|---------|---------|
| فحص انتهاء العضوية | يومي | تحديد العضويات المنتهية |
| تجديد تلقائي | يومي | تجديد العضويات المؤهلة |
| تذكيرات الانتهاء | يومي | إرسال بريد قبل 7 أيام ويوم |
| تنظيف الأرصدة المنتهية | يومي | تحديد المعاملات المنتهية |
| كشف عدم الحضور | كل ساعة | تصنيف بعد فترة سماح ساعتين |
| خروج تلقائي | كل ساعة | تسجيل خروج الحجوزات المتأخرة |
| توليد التوثيق | 02:00 يومياً | إعادة توليد التوثيق التلقائي |

تأكد من تشغيل العمال:
```bash
bench doctor
supervisorctl restart all
```

## النسخ الاحتياطي والصيانة

### النسخ الاحتياطي

```bash
# يدوي
bench --site your-site backup --with-files

# مجدول (أضف إلى crontab)
0 2 * * * cd /home/frappe/frappe-bench && bench --site your-site backup --with-files
```

### إدارة الذاكرة المؤقتة

```bash
bench --site your-site clear-cache
bench --site your-site clear-website-cache
```

### التحديث

```bash
cd apps/arkspace && git pull
bench --site your-site migrate
bench build --app arkspace
bench restart
```
