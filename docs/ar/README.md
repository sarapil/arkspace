## حول ARKSpace

ARKSpace هو تطبيق شامل لإدارة مساحات العمل المشتركة، مبني أصلاً على إطار عمل Frappe مع تكامل عميق مع ERPNext. يوفر للمشغلين كل ما يحتاجونه لإدارة مساحاتهم — من جذب العملاء إلى الفوترة — في منصة واحدة متكاملة.

## البدء السريع

```bash
# التثبيت
bench get-app https://github.com/arkan/arkspace.git
bench --site your-site install-app arkspace
bench --site your-site migrate
bench build --app arkspace

# الوصول
# لوحة التحكم: http://your-site/desk/arkspace
# بوابة الأعضاء: http://your-site/member-portal
# مخطط الطوابق: http://your-site/desk/arkspace/floor-plan
```

## الميزات الرئيسية

| الوحدة | الوصف |
|--------|-------|
| **المساحات** | 6 أنواع مساحات، مخطط طوابق تفاعلي، إشغال فوري |
| **الحجوزات** | بالساعة/يومي/شهري، تسجيل دخول/خروج تلقائي |
| **العضويات** | خطط مع محافظ ائتمان، تجديد تلقائي |
| **إدارة العملاء** | خط مبيعات، جدولة جولات، تحويل بنقرة واحدة |
| **العقود** | قوالب ثنائية اللغة، مستندات قانونية، إيصالات |
| **التدريب** | وحدات، جلسات، شارات، تتبع التقدم |
| **الفوترة** | تكامل أصلي مع ERPNext، فوترة تلقائية |
| **التحليلات** | 3 تقارير، لوحة معلومات بالبطاقات والرسوم البيانية |
