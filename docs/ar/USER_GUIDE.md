## البدء

### تسجيل الدخول الأول

1. افتح المتصفح وانتقل إلى رابط ARKSpace الخاص بك
2. سجّل الدخول بالبريد الإلكتروني وكلمة المرور
3. ستظهر لك **لوحة ARKSpace** مع بطاقات المعلومات

### التنقل

| المنطقة | الرابط | الوصف |
|---------|--------|-------|
| لوحة التحكم | `/desk/arkspace` | واجهة التطبيق الرئيسية |
| مخطط الطوابق | `/desk/arkspace/floor-plan` | نظرة عامة مرئية على المساحات |
| ARK Live | `/desk/arkspace/ark-live` | مراقبة المساحات بالوقت الفعلي |
| بوابة الأعضاء | `/member-portal` | خدمة ذاتية للأعضاء |

## حجز مساحة

### من لوحة التحكم

1. انتقل إلى **المساحات > حجز مساحة > + جديد**
2. اختر **المساحة** (ابحث بالاسم أو النوع)
3. اختر **العضو** (العميل)
4. اختر **نوع الحجز**: بالساعة، يومي، أو شهري
5. حدد تاريخ ووقت **البداية** و **النهاية**
6. أضف **خدمات إضافية** إذا لزم الأمر
7. اضغط **حفظ** ثم **إرسال**
8. تتغير الحالة إلى **مؤكد** ✅

### من مخطط الطوابق

1. انتقل إلى **ARKSpace > مخطط الطوابق**
2. المساحات الخضراء متاحة — اضغط على واحدة
3. املأ نافذة الحجز السريع
4. اضغط **حجز** — تم!

### حالات الحجز

```
قيد الانتظار → مؤكد → تم تسجيل الدخول → تم تسجيل الخروج
                    ↘ ملغي
                    ↘ عدم حضور
```

## تسجيل الدخول والخروج

### تسجيل دخول فردي

1. افتح حجز المساحة
2. اضغط زر **تسجيل الدخول** (يظهر للحجوزات المؤكدة)
3. تتغير حالة المساحة إلى **مشغولة**
4. يتم تحديث مخطط الطوابق وARK Live بالوقت الفعلي

### تسجيل خروج فردي

1. افتح حجز المساحة
2. اضغط زر **تسجيل الخروج**
3. تعود المساحة إلى **متاحة**

### العمليات الجماعية (الاستقبال / العمليات)

1. انتقل إلى **المساحات > حجز مساحة** — عرض القائمة
2. حدد عدة حجوزات باستخدام مربعات الاختيار
3. استخدم قائمة **الإجراءات**:
   - **تسجيل دخول جماعي**
   - **تسجيل خروج جماعي**
   - **إلغاء جماعي**
   - **عدم حضور جماعي**

### إجراءات تلقائية

- **كشف عدم الحضور**: الحجوزات غير المسجلة خلال ساعتين يتم تصنيفها تلقائياً
- **خروج تلقائي**: الحجوزات المتأخرة يتم تسجيل خروجها تلقائياً

## العضويات

### عرض الخطط

انتقل إلى **العضويات > خطة العضوية** لعرض الخطط المتاحة.

تشمل كل خطة:
- نوع الخطة (مكتب مشترك، مكتب مخصص، مكتب خاص، إلخ)
- أسعار شهرية / ربع سنوية / سنوية
- مزايا مضمنة (ساعات، رصيد، ضيوف، غرف اجتماعات، طباعة، تخزين)

### إنشاء عضوية

1. انتقل إلى **العضويات > عضوية > + جديد**
2. اختر **العضو** (العميل)
3. اختر **خطة العضوية**
4. اختر **دورة الفوترة** (شهري/ربع سنوي/سنوي)
5. حدد **تاريخ البدء**
6. فعّل **التجديد التلقائي** إذا رغبت
7. **احفظ** و **أرسل**
8. يتم إنشاء **محفظة الرصيد** تلقائياً

### محفظة الرصيد

لكل عضو محفظة رصيد:

| الحقل | الوصف |
|-------|-------|
| إجمالي الرصيد | جميع الأرصدة المستلمة |
| الرصيد المستخدم | الأرصدة المستهلكة |
| الرصيد المتاح | الرصيد الحالي |

### العمليات التلقائية

- **فحص انتهاء الصلاحية**: مهمة يومية تحدد العضويات المنتهية
- **التجديد التلقائي**: مهمة يومية تجدد العضويات المؤهلة
- **التذكيرات**: إشعارات بالبريد قبل 7 أيام ويوم واحد من الانتهاء

## بوابة الأعضاء

### الوصول

انتقل إلى `/member-portal` بعد تسجيل الدخول.

### أقسام لوحة المعلومات

| القسم | ما تراه |
|-------|---------|
| العضويات النشطة | بطاقات العضوية الحالية |
| الحجوزات القادمة | الحجوزات المستقبلية |
| الحجوزات الأخيرة | سجل الحجوزات |
| الإحصائيات | إجمالي الحجوزات، إجمالي المدفوعات |

### إجراءات الخدمة الذاتية

- **حجز مساحة** — اختر النوع والتواريخ وأكد
- **عرض الحجوزات** — الحجوزات القادمة والسابقة
- **إلغاء حجز** — من قائمة الحجوزات القادمة
- **عرض العضوية** — تفاصيل الخطة ورصيد المحفظة

## إدارة العملاء والمبيعات

### إدارة العملاء المحتملين

#### إنشاء عميل محتمل

1. انتقل إلى **إدارة العملاء > عميل محتمل > + جديد**
2. املأ: الاسم، البريد الإلكتروني، الهاتف
3. اختر **المصدر**: موقع إلكتروني، زيارة، إحالة، وسائل التواصل، فعالية، شريك
4. اختر الخطة المهتم بها، الميزانية، حجم الفريق (اختياري)
5. **احفظ**

#### خط المبيعات

```
جديد → تم التواصل → جولة مجدولة → تفاوض → تم التحويل
                                            ↘ خسارة
```

#### جدولة جولة

1. افتح سجل العميل المحتمل
2. اضغط **جدولة جولة**
3. حدد التاريخ والوقت والمدة
4. يتم إنشاء جولة مساحة العمل تلقائياً
5. تتغير حالة العميل إلى **جولة مجدولة**

#### التحويل إلى عميل

1. افتح العميل المحتمل
2. اضغط **تحويل لعميل**
3. يتم إنشاء سجل عميل في ERPNext
4. تتغير الحالة إلى **تم التحويل**
5. يمكنك الآن إنشاء عضوية للعميل الجديد

## العقود

### إنشاء عقد

1. انتقل إلى **العقود > عقد عضو > + جديد**
2. املأ: العضو، نوع العقد، القالب، التواريخ، القيمة
3. اضغط **تطبيق القالب** لتعبئة الشروط
4. أرفق **المستندات القانونية** (هوية، جواز سفر، إلخ)
5. احصل على **توقيع العضو**
6. **احفظ** و **أرسل**

### المستندات القانونية

الأنواع المدعومة: هوية وطنية، جواز سفر، سجل تجاري، بطاقة ضريبية، وغيرها.

### إيصالات الدفع

طرق الدفع: نقدي، تحويل بنكي، بطاقة ائتمان، شيك، دفع إلكتروني، محفظة.

## التدريب

### تصفح الكتالوج

1. انتقل إلى **التدريب > وحدة تدريبية**
2. فلتر حسب الفئة (تقني، أعمال، إبداعي، صحة، مجتمع، تهيئة)
3. فلتر حسب المستوى (مبتدئ، متوسط، متقدم)

### التسجيل في جلسة

1. ابحث عن **جلسة تدريبية** قادمة
2. اضغط **التسجيل**
3. يتم تتبع تقدمك في **تقدم تدريب المستخدم**

### الحصول على شارات

يتم منح الشارات تلقائياً:

| المستوى | الإنجاز |
|---------|---------|
| 🥉 برونزي | إكمال أساسي |
| 🥈 فضي | إنجاز متوسط |
| 🥇 ذهبي | إتقان متقدم |
| 💎 بلاتيني | أداء استثنائي |
