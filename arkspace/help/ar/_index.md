# مساعدة ARKSpace

مرحباً بك في ARKSpace — منصتك الشاملة لإدارة مساحات العمل المشتركة.

## البدء

بعد تسجيل الدخول، ستجد مساحة عمل ARKSpace على لوحتك مع:
- **لوحة معلومات** تعرض الحجوزات والعضويات وإحصائيات الإيرادات
- **إجراءات سريعة** للمهام الشائعة
- **تنقل** لجميع الوحدات

## الوحدات

- **المساحات** — إدارة أنواع المساحات والخدمات والمساحات الفردية
- **الحجوزات** — إنشاء وإدارة حجوزات المساحات مع تسجيل الدخول/الخروج
- **العضويات** — خطط اشتراك مع محافظ ائتمان
- **إدارة العملاء** — خط مبيعات وجدولة جولات
- **العقود** — مستندات قانونية وإدارة عقود
- **التدريب** — وحدات تعلم مع شارات وتتبع تقدم

## الصفحات الرئيسية

- **مخطط الطوابق** (`/desk/arkspace/floor-plan`) — مخطط طوابق تفاعلي
- **ARK Live** (`/desk/arkspace/ark-live`) — مراقبة المساحات بالوقت الفعلي

## تحتاج مساعدة؟

- راجع [دليل المستخدم](/docs/ar/USER_GUIDE.md) للتعليمات التفصيلية
- تواصل مع مسؤول النظام لمشاكل الوصول
- البريد الإلكتروني: dev@arkspace.io
