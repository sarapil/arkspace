# عميل محتمل

العميل المحتمل يتتبع عميلاً محتملاً عبر خط المبيعات من أول تواصل إلى التحويل.

## خط المبيعات

```
جديد → تم التواصل → جولة مجدولة → تفاوض → تم التحويل
                                          ↘ خسارة
```

## الحقول الرئيسية

- **اسم العميل**: الاسم الكامل للعميل المحتمل
- **البريد الإلكتروني / الهاتف**: معلومات الاتصال
- **المصدر**: موقع إلكتروني، زيارة، إحالة، وسائل التواصل، فعالية، شريك
- **الحالة**: الموقع الحالي في خط المبيعات
- **الخطة المهتم بها**: خطة العضوية المرغوبة
- **الميزانية**: نطاق ميزانية العميل
- **حجم الفريق**: عدد المكاتب المطلوبة

## الإجراءات

- **جدولة جولة**: ينشئ جولة مساحة عمل ويحدث الحالة
- **تحويل لعميل**: ينشئ سجل عميل في ERPNext، يحدد كـ "تم التحويل"

## نصائح

- حدّث الحالة كلما تقدمت في خط المبيعات
- جدول الجولات مبكراً — تحسّن معدل التحويل بشكل كبير
- بعد التحويل، أنشئ عضوية للعميل الجديد
