# العضوية

العضوية تمثل اشتراك العضو في خطة مع مزايا مضمنة وتخصيص رصيد.

## سير العمل

```
مسودة → نشط → منتهي
          ↘ معلق
          ↘ ملغي
```

## الحقول الرئيسية

- **العميل**: العضو المشترك
- **خطة العضوية**: الخطة المختارة بالتسعير والمزايا
- **دورة الفوترة**: شهري، ربع سنوي، أو سنوي
- **تاريخ البداية/النهاية**: فترة الاشتراك
- **تجديد تلقائي**: تبديل للتجديد التلقائي
- **الحالة**: حالة العضوية الحالية

## الميزات التلقائية

- **محفظة الرصيد**: تُنشأ تلقائياً عند الإرسال مع الرصيد المخصص
- **التجديد التلقائي**: مهمة يومية تجدد العضويات المؤهلة قبل الانتهاء
- **تذكيرات الانتهاء**: إشعارات بالبريد قبل 7 أيام ويوم واحد
- **الفوترة**: فاتورة مبيعات تلقائية عند الإرسال، إشعار دائن عند الإلغاء

## لوحة العضو

توفر واجهة لوحة العضو نظرة موحدة على:
- العضويات النشطة
- الحجوزات القادمة والأخيرة
- رصيد محفظة الائتمان
- إحصائيات الاستخدام
