# حجز مساحة

حجز المساحة يسجل حجز مساحة عمل مشتركة لعضو.

## سير العمل

```
قيد الانتظار → مؤكد → تم تسجيل الدخول → تم تسجيل الخروج
                    ↘ ملغي
                    ↘ عدم حضور
```

## الحقول الرئيسية

- **المساحة**: مساحة العمل المشتركة المحجوزة
- **العميل**: العضو الذي يقوم بالحجز
- **نوع الحجز**: بالساعة، يومي، أو شهري
- **تاريخ/وقت البداية والنهاية**: فترة الحجز
- **الحالة**: حالة الحجز الحالية
- **خدمات إضافية**: خدمات اختيارية إضافية

## الإجراءات

- **إرسال**: يؤكد الحجز (ينشئ فاتورة مبيعات تلقائياً)
- **تسجيل الدخول**: يسجل حضور العضو، يحدث المساحة إلى مشغولة
- **تسجيل الخروج**: يكمل الحجز، يحرر المساحة
- **إلغاء**: يلغي الحجز (ينشئ إشعار دائن)

## نصائح

- استخدم صفحة **مخطط الطوابق** للحجز المرئي
- **العمليات الجماعية** متاحة في عرض القائمة لكفاءة الاستقبال
- يتم كشف عدم الحضور تلقائياً بعد ساعتين
- يتم تسجيل خروج الحجوزات المتأخرة تلقائياً
