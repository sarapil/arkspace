# 📖 Glossary — قاموس المصطلحات

| English | العربية | Context |
|---------|---------|---------|
| Amenity | مرفق / ميزة | Feature available in a space (WiFi, parking…) |
| Booking | حجز | Reservation of a space for a time period |
| Branch | فرع | Physical co-working location |
| Check-in | تسجيل دخول | Member arriving at their booked space |
| Check-out | تسجيل خروج | Member leaving their booked space |
| Co-working Space | مساحة عمل مشترك | A bookable unit (desk, office, room) |
| Credit | رصيد | Prepaid balance in member wallet |
| Credit Wallet | محفظة الرصيد | Member's prepaid account |
| Dashboard | لوحة التحكم | Overview screen with KPIs |
| Hot Desk | مكتب مشترك | Shared desk, first-come-first-served |
| IoT | إنترنت الأشياء | Internet of Things (ARKAMOR sensors) |
| KPI | مؤشر أداء | Key Performance Indicator |
| Lead | عميل محتمل | Prospective member |
| Membership | عضوية | Active subscription contract |
| Membership Plan | خطة العضوية | Template defining plan benefits & pricing |
| Occupancy Rate | نسبة الإشغال | Percentage of spaces currently in use |
| Pipeline | خط المبيعات | Sales funnel stages |
| Private Office | مكتب خاص | Dedicated office for one company |
| RevPAD | العائد لكل مكتب | Revenue Per Available Desk |
| Space Type | نوع المساحة | Category of spaces |
| Tour | جولة تعريفية | Guided visit for prospective members |
| Virtual Address | عنوان افتراضي | Business address service |
| Workspace | مساحة عمل | General term or Frappe desk workspace |
